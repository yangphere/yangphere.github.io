centos 6.4升级默认python2.6到2.7

· 2022-07-19 · # linux # python

官网下载

wget https://www.python.org/ftp/python/2.7.11/Python-2.7.11.tar.xz

解压

xxx.tar.xz的解压方式，xz -d xxx得到xxx.tar包，还得使用tar xvf xxx解压。xxx.tar.xz包的压缩过程是tar cvf xxx.tar xxx/和 xz -z xxx.tar.xz

xz -d Python-2.7.11.tar.xz
tar xvf Python-2.7.11.tar

安装

./configure
make
make install

修改系统默认版本

cd /usr/bin/
mv python python2.6.6
ln -s /usr/local/bin/python2.7 python

查看python版本，显示版本升级成功。

# python -V
Python 2.7.11

修改yum命令依赖python2.6版本

打开yum脚本文件：vim /usr/bin/yum
把 #!/usr/bin/python 修改为 #!/usr/bin/python2.6
